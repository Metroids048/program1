# 页面映射

## 建议的小程序页面

1. 岗位台
2. 实时助手
3. 模拟面试配置
4. 模拟面试答题
5. JD 分析
6. 问题库
7. 简历
8. 记录复盘列表
9. 记录复盘详情

## 当前桌面端来源

- `/` -> 岗位台
- `/live` -> 实时助手
- `/mock` -> 模拟面试配置 + 答题
- `/jd` -> JD 分析
- `/questions` -> 问题库
- `/resume` -> 简历
- `/records` -> 记录列表
- `/records/:id` -> 记录详情

## 小程序拆分建议

- 模拟面试的“进入前配置弹窗”应单独变成一个配置页面。
- 记录详情保持独立页面，不与列表复用同一视图。
