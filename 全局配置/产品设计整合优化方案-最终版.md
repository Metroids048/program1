## 一、设计系统规范

### 1.1 颜色系统（CSS 变量）

```css
/* 背景层级 */
--bg-page:          #F7F7F5;   /* 页面主背景，暖白 */
--bg-card:          #FFFFFF;   /* 卡片/面板背景 */
--bg-surface:       #F0EEE8;   /* 输入框、语音识别区等次级表面 */
--bg-topbar:        #1A1A18;   /* 实时助手深色状态栏 */

/* 文本层级 */
--text-primary:     #1A1A18;   /* 主文本 */
--text-secondary:   #6B6B67;   /* 次文本 */
--text-tertiary:    #9E9E99;   /* 辅助文本/占位符 */
--text-disabled:    #C4C2BC;   /* 禁用文本 */

/* 边框 */
--border-default:   #E8E6E0;   /* 默认 0.5px 边框 */
--border-light:     #F0EEE8;   /* 分割线 */

/* 功能色 */
--green:            #2D9D6B;   /* 主色：行动/激活/完成/模型生成 */
--green-hover:      #1F7A52;   /* hover/pressed */
--green-bg:         rgba(45,157,107,0.08);   /* 绿色浅底 */
--green-border:     rgba(45,157,107,0.25);   /* 绿色边框 */

--red:              #C4411C;   /* 录音中/风险提醒/错误 */
--red-bg:           rgba(196,65,28,0.08);
--red-glow:         rgba(196,65,28,0.12);    /* 录音按钮光晕 */

--blue:             #2B5BA8;   /* 面试中状态/信息标签 */
--blue-bg:          rgba(43,91,168,0.08);

--orange:           #C4691C;   /* 证据引用/本地练习/重要标注 */
--orange-bg:        rgba(196,105,28,0.06);

--purple:           #534AB7;   /* 模拟练习相关元素 */
--purple-bg:        rgba(83,74,183,0.04);    /* 题目卡背景 */
--purple-tag-bg:    rgba(83,74,183,0.10);    /* 题型标签背景 */
```

### 1.2 字号体系（Type Scale）

| Token | 字号 | 字重 | 行高 | 使用场景 |
|-------|------|------|------|----------|
| `text-h1` | 20px | 500 | 1.4 | 页面唯一标题 |
| `text-h2` | 16px | 500 | 1.4 | 模块标题、提词卡开场句 |
| `text-h3` | 14px | 500 | 1.5 | 卡片内小标题 |
| `text-body` | 14px | 400 | 1.7 | 通用正文、对话消息、输入框文字 |
| `text-body-sm` | 13px | 400 | 1.7 | 次要正文、提词卡要点、状态栏文字 |
| `text-caption` | 12px | 400 | 1.6 | 辅助信息、证据内容、风险提醒 |
| `text-micro` | 11px | 400 | 1.5 | 标签、徽章、时间戳占位符 |

**强制规则：**
- 全站仅此 7 级字号，任何组件不得出现 10px / 15px / 18px / 22px 等非标准字号
- H1 每页只出现一次，其余标题从 H2 起步
- 正文（text-body）不加粗；加粗仅用于标题和关键词
- 标签/徽章统一 11px（text-micro），全局不混用 10px / 12px 标签

### 1.3 间距系统（4px 基准）

| Token | 值 | 使用场景 |
|-------|-----|----------|
| `space-xs` | 8px | 组件内紧凑间距、图标与文字间距 |
| `space-sm` | 12px | 组件内标准间距、标签间距 |
| `space-md` | 16px | 卡片内边距、组件间标准间距 |
| `space-lg` | 24px | 区块间距、页面内容区上下间距 |
| `space-xl` | 32px | 大区块分隔 |
| `space-2xl` | 48px | 页面顶部留白 |

### 1.4 圆角

| Token | 值 | 使用场景 |
|-------|-----|----------|
| `radius-sm` | 4px | 标签、徽章、迷你按钮 |
| `radius-md` | 8px | 按钮、输入框、标准卡片 |
| `radius-lg` | 10px | 大卡片、面板、弹窗 |
| `radius-full` | 50% | 圆形录音按钮 |

### 1.5 边框规范

- 默认边框：`0.5px solid var(--border-default)` — 绝不用 1px 实线
- 卡片激活左边框：`3px solid [状态色]` — 仅用于选中态标记
- 分割线：`0.5px solid var(--border-light)`

### 1.6 阴影

| Token | 值 | 使用场景 |
|-------|-----|----------|
| `shadow-sm` | `0 1px 3px rgba(0,0,0,0.04)` | 卡片默认 |
| `shadow-md` | `0 4px 12px rgba(0,0,0,0.05)` | 悬浮卡片、下拉 |
| `shadow-recording` | `0 0 0 8px var(--red-glow)` | 录音中按钮光晕 |

**规则：** 阴影仅用于浮层交互（下拉、弹窗），静态卡片不用阴影。

### 1.7 字体

- 标题/展示：Noto Serif SC（衬线）
- 正文/UI：Noto Sans SC（无衬线）
- 代码/数据：ui-monospace（等宽，用于 JD 文本、计时器）

### 3.3 对话区域详细规范

**容器：**
- `max-width: 680px`，水平居中（`margin: 0 auto`）
- `min-height: 120px`，高度自适应内容
- 背景：`--bg-surface`（`#F0EEE8`），与外层 `--bg-page` 区分
- 圆角：`--radius-lg`（10px）
- 内边距：`--space-md`（16px）

**消息气泡：**
- AI 消息：白色背景卡片，圆角 8px，内边距 12px 14px，左对齐
  - 前置小 AI图标（16px，`--green`），与文字间距 8px
  - 文字 14px / 400 / 行高 1.7 / `--text-primary`
- 用户消息：`--green-bg` 浅色背景，右对齐
  - 文字同规格，`--text-secondary`
- 消息间距：`--space-sm`（12px）

**对话折叠规则：**
- 默认显示最近 3 条消息
- 超过 3 条时，第 3 条上方显示「查看完整对话 ›」（12px，`--text-tertiary`），点击展开
- 展开后最大高度 400px，超出滚动

**输入框：**
- 置于对话区域底部
- 全宽，最小高度 48px，自适应（max-height 120px）
- 背景 `--bg-card`（白色），与对话区底色区分
- 右侧 [发送] 按钮：主色胶囊，高度 30px，文字 "发送" 12px
- placeholder："描述你想要的岗位，比如：字节跳动 AI产品运营实习..."
- 下方推荐 chips（仅首次无对话时显示）：
  - 3 个预设提示词 chips（如「产品经理」「前端开发」「数据分析」）
  - 11px / 灰色边框 / 圆角 20px，点击填入输入框
  - 水平排列，间距 8px

### 3.4 岗位卡详细规范

**卡片容器：**
- 宽度 100%（占满内容区），最大 680px（与对话区间宽）
- 高度：固定 56px
- 背景：`--bg-card`（白色）
- 圆角：`--radius-md`（8px）
- 内边距：10px 16px
- 布局：flex，`align-items: center`，`justify-content: space-between`
- 卡片间距：`--space-xs`（8px）

**状态左边框：**
- 3px solid，位于卡片最左侧（通过 `border-left` 或 `::before` 伪元素）
- 进行中（未完成面试）：`--green`
- 面试中（正在实时助手）：`--blue`
- 已完成（有复盘记录）：`--text-disabled`（`#C4C2BC`）

**左侧内容区（flex: 1）：**
- 第一行（14px / 500 / `--text-primary`）：
  - `公司名 · 岗位名`
  - 右侧紧跟状态徽章（仅在"面试中"时显示蓝色徽章，11px）
- 第二行（12px / 400 / `--text-secondary`）：
  - `事业部 · 城市 · 导入日期`

**右侧操作区（flex-shrink: 0）：**
- 进行中 → [继续模拟]：Secondary 按钮风格，13px / 500
- 面试中 → [进入助手]：Primary 按钮风格（绿底白字），13px / 500
- 已完成 → [查看复盘]：Ghost 按钮风格，13px / `--text-tertiary`

**卡片交互：**
- hover：背景轻微变深（`--bg-surface`）
- 点击卡片主体（非按钮区域）→ 同按钮行为
- 激活/选中态（当前正在操作的岗位）：左边框变为强调色 + 背景 `--green-bg`

### 3.5 "新增岗位"触发

**触发方式：** 点击右上角 [+ 新增岗位] 按钮

**形式：** 对话框聚焦到输入框，用户直接在首页输入 JD 即可。不需要单独弹出 Drawer（当前实现就是这样，保持不变）。

### 3.6 移除项

- **移除右侧 Context Panel**（匹配度百分比、资料数、职位切换下拉等）—— 这些信息属于资料库各页面，不属于首页
- **移除大段 JD 诊断信息** —— 解析结果只在对话流中以紧凑卡片形式出现，不占满整个页面

### 3.7 空状态

当没有任何岗位时：
- 对话区域居中显示
- AI 首条消息保留
- 底部「我的岗位」区域隐藏
- 输入框聚焦状态（自动 focus）

**自动生成模式：**

5. 提词卡区显示 Skeleton 加载态（灰色骨架屏）
6. 请求返回 → 卡片淡入（300ms），AI 徽章变绿"● 模型生成"

**手动确认模式：**
1. 同上步骤 1-3
2. 停止后不自动生成，显示 [生成提词卡] 按钮
3. 用户点击按钮才触发请求

**生成超时（10s）：**
- 降级到本地规则生成
- AI 状态徽章显示橙色"● 本地练习"
- 提示文字："后端未响应，已使用本地规则生成"（12px / `--text-tertiary`，显示 3s 后消失）

**结束面试：**
1. 点击 [结束助手] → 弹出确认弹窗
2. 弹窗内容：岗位名 + 面试时长 + 处理题数 + [确认结束] / [继续面试]

## 八、全局交互与状态规范

### 8.1 AI 状态徽章（全局统一）

所有展示 AI 生成内容的地方（提词卡、追问气泡、报告）必须显示状态徽章：

```
● 模型生成  — 绿色圆点 + 绿色文字，静止
● 本地练习  — 橙色圆点 + 橙色文字，静止
● 生成中... — 灰色圆点 + 灰色文字，圆点旋转动画（600ms/圈）
```

- 位置：卡片的右上角
- 样式：11px，内边距 2px 8px，圆角 4px
- 对应背景：绿→`--green-bg`，橙→`--orange-bg`，灰→`--bg-surface`
- 过渡：颜色切换 150ms ease

### 8.2 语音转写三态

| 状态 | 文字颜色 | 光标 | 可编辑 |
|------|---------|------|--------|
| interim（识别中） | `#9E9E99` | 末尾闪烁 | 否 |
| final（已确认） | `#1A1A18` | 无 | 是 |
| editable（编辑中） | `#1A1A18` | textarea 光标 | 是 |

- 停止录音后文字不清空
- 只有点击 [清空重录] 才清空
- 切换到可编辑状态时显示 "已识别 · 可编辑"（11px / `--text-tertiary`）

### 8.3 加载状态

**Skeleton 骨架屏（用于提词卡生成中）：**
- 矩形灰色块（`--bg-surface`），带 shimmer 动画
- 六区块骨架：6 条不同宽度的灰色条（宽度百分比递减：100% / 85% / 70% / 60% / 90% / 50%）

**Spinner（用于按钮加载中）：**
- 12px 旋转圆环，`--green`

**文本提示（用于生成超时）：**
- 12px / `--text-tertiary`，"后端未响应，已使用本地规则生成"
- 显示 3s 后自动消失

### 8.4 过渡动画

| 场景 | 动画 | 时长 |
|------|------|------|
| 提词卡切换 | 旧卡 translateY 下移 + 新卡 opacity 淡入 | 200ms / 300ms |
| AI 状态徽章变化 | 颜色 + 文字 dissolve | 150ms |
| 生成中旋转 | rotate 360° 循环 | 600ms/圈 |
| 追问气泡出现 | opacity 0→1 + translateY 4px→0 | 200ms |
| 折叠/展开 | max-height 过渡 | 250ms |
