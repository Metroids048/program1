# 人工浏览器验收清单

> 执行环境：Cursor 或任何浏览器，`http://127.0.0.1:5173/`
> Codex Desktop / Cursor / 人工浏览器都可以执行此清单；若某个浏览器工具崩溃，记录工具故障并改用外部 Playwright 或系统浏览器兜底（`npm run test:browser-flow`）。

## 前置条件

1. 终端执行 `npm run dev` 启动前端开发服务器
2. 新开终端执行 `npm run server` 启动后端（或 `npm run dev:full`）
3. 确认 `http://127.0.0.1:5173/` 可访问

---

## 一、1280×720 首页首屏

- [ ] 设置浏览器窗口为 1280×720
- [ ] 访问首页 `/`
- [ ] 确认：主标题可见（"真实 JD intake" 或等效标题）
- [ ] 确认：JD intake 输入框可见且可编辑
- [ ] 确认：右侧无大面积无意义空白
- [ ] 确认：无控制台 error
- [ ] 输入测试 JD："公司：腾讯 | 岗位：AI 产品经理\n负责用户访谈、SQL 分析和增长复盘" → 点击发送
- [ ] 确认：出现追问/缺失字段提示
- [ ] 确认：岗位卡出现（显示公司名和岗位名）
- [ ] 点击"保存岗位" → 确认岗位卡仍可见
- [ ] 点击"进入模拟配置" → 确认跳转到 `/mock`

**结果：** ⬜ 可发布 / ⬜ 可发布但需修复 / ⬜ 不可发布

---

## 二、1280×720 实时助手

- [ ] 导航到"实时助手"
- [ ] 确认：问题输入区可见
- [ ] 确认："手动确认"和"自动生成"按钮存在
- [ ] 切换到"手动确认"模式
- [ ] 点击"开始听取"（如果浏览器不支持语音，应显示降级提示）
- [ ] 在文本框中输入："请介绍一个你做过的 AI 产品项目"
- [ ] 点击"停止听取"（如果有语音）/ 直接编辑文本
- [ ] 确认：已输入文本保留，未被清空
- [ ] 点击"生成提词卡"
- [ ] 确认：提词卡出现，包含策略/开场白/要点/风险提醒/追问预测
- [ ] 确认：若无模型，显示"本地练习模式"相关提示
- [ ] 点击"记录到面试资料" → 确认反馈
- [ ] 点击"结束" → 确认结束对话框出现

**结果：** ⬜ 可发布 / ⬜ 可发布但需修复 / ⬜ 不可发布

---

## 三、1280×720 模拟面试

- [ ] 导航到"模拟面试"
- [ ] 确认：开始对话框出现（"开始模拟面试"）
- [ ] 选择配置（上级/压力面/手动提交） → 点击"开始"
- [ ] 确认：面试官问题出现在对话区
- [ ] 确认：顶部状态栏可见（显示面试进度或状态）
- [ ] 确认：底部语音控制可见（如果支持）
- [ ] 点击"生成提词卡" → 确认提词卡侧栏出现
- [ ] 在回答区输入："我负责过校园增长项目，通过用户访谈和漏斗分析，把首单转化率从 12% 提升到 19%。"
- [ ] 点击"提交当前回答"
- [ ] 确认：出现追问（新问题）
- [ ] 确认：出现即时反馈（如"结构清晰"等）
- [ ] 再次输入回答 → 点击"结束"
- [ ] 在结束对话框中选择"保存并结束"
- [ ] 确认：跳转到面试记录页，报告可见

**结果：** ⬜ 可发布 / ⬜ 可发布但需修复 / ⬜ 不可发布

## 四、390px 移动端

- [ ] 在浏览器 DevTools 中切换到移动端视图，设置宽度 390px
- [ ] 首页：确认输入框不横向溢出
- [ ] 首页：确认按钮不重叠
- [ ] 导航到"实时助手"：确认文本区和按钮在屏幕内
- [ ] 导航到"模拟面试"：确认对话区、输入框、控制按钮不溢出
- [ ] 导航到"问题库"：确认资料卡片不横向溢出
- [ ] 导航到"简历"：确认区块和输入框在屏幕内
- [ ] 导航到"面试记录"：确认列表项不溢出

**结果：** ⬜ 可发布 / ⬜ 可发布但需修复 / ⬜ 不可发布

## 五、控制台与交互反馈

- [ ] 打开浏览器 DevTools Console
- [ ] 依次导航所有 7 个路由：岗位台、实时助手、模拟面试、JD 解析、问题库、简历、面试记录
- [ ] 确认：控制台无红色 error
- [ ] 确认：切换路由时无白屏闪烁
- [ ] 在"实时助手"中：快速切换"手动确认"↔"自动生成"，确认无异常
- [ ] 在"账户与数据"弹窗中：导出 → 确认下载 JSON 文件
- [ ] 导入刚导出的文件 → 确认提示"导入成功"
- [ ] 导入一个非法 JSON 文件 → 确认提示"导入失败"

**结果：** ⬜ 可发布 / ⬜ 可发布但需修复 / ⬜ 不可发布
