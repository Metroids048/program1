# 历史资料归档策略

更新时间：2026-06-19

## 原则

- 历史方案不直接删除。
- 现行判断写入 `docs/current/`。
- 旧方案、旧评估、历史快照继续保留在仓库原位置或后续迁入 `docs/archive/`。

## 当前主参考

- `参考资料/产品设计整合优化方案-最终版.md`

## 当前已被用户显式覆盖的点

- 侧栏在实时助手中自动收起：不再采用
- 问题库视觉重点不足：已按资料底座重排
- 简历页右侧非标准对话样式：不再采用

## 使用建议

- 需要当前真实能力时，只看 `docs/current/`
- 需要追溯历史方案时，再回看 `参考资料/` 与归档说明
