# 真实浏览器补验清单 — 2026-07-06

## 使用方式

在 Cursor Browser / Playwright MCP 或人工浏览器中打开 `http://127.0.0.1:5173/`。本清单只用于渲染层、点击体验、控制台和响应式检查；Codex Desktop 不执行这些步骤。

每项结果填写 `Pass / Fail / Blocked`，Fail 必须记录截图或录屏、复现步骤、浏览器控制台错误和影响等级。

## 前置数据

- 使用 `测试用/AI产品经理.pdf` 作为简历上传样本
- 使用 `测试用/项目材料/AI文本功能.docx` 作为项目资料上传样本

```text
负责用户访谈、SQL 分析、增长复盘和 AI 功能落地。
要求熟悉数据分析、产品协同和项目推进。
```

## 判定规则

- P0：白屏、核心链路 500、登录/账号串数据、记录丢失、无法保存报告
- P1：功能可用但误导用户，例如 fallback 伪装成功、上传无状态、历史提词卡丢失
- P2：轻微文案、布局、间距或提示问题
