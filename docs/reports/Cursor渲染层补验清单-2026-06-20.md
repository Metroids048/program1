# Cursor 渲染层补验清单

更新时间：2026-06-20

## 1. 使用边界

- 本清单仅用于 Cursor 或人工浏览器补验。
- Codex Desktop 不执行内置 Browser / IAB / Chrome / Computer Use 验收。
- 访问入口只输出本地地址与日志位置，不自动弹浏览器。
- 交付说明需明确标注：`Codex 无渲染层验收`。

## 2. 桌面端补验

分辨率：`1280x720`

### 首页

检查项：

1. JD 卡可见，且不会挤压主输入区
2. 配置入口可见，点击后不遮挡主 CTA
3. 首屏只保留主标题、对话输入、岗位卡、主要 CTA、最小右侧上下文
4. 无大段诊断信息堆在首屏

### 实时助手

检查项：

1. 文本输入可直接触发题词卡生成
2. 语音输入开始、停止后，已识别文本仍保留
3. 已识别文本可编辑，编辑后可重新生成题词卡
4. SSE 过程中 stage / delta / card 的 UI 状态无异常闪烁或卡死
5. 顶部状态、题词卡区、底部语音控制同时可见

### 模拟面试

检查项：

1. 首题正常进入
2. 回答一题后能生成追问或下一题
3. 顶部状态栏、对话区、底部语音控制、当前题词卡同时可见
4. 结束面试后能保存报告并进入记录页
5. 若后端不可用，必须明确显示本地练习模式，而不是伪装成功

### 记录页

检查项：

1. 时间轴顺序正确
2. Transcript 展开/收起可用
3. 报告指标卡层级清晰，不出现卡片套卡片堆叠感
4. `synthetic evidence` 或练习推断标签可见时，文案不误导为真实经历

## 3. 移动端补验

分辨率：`390px`

检查项：

1. 导航、按钮、卡片、题词卡、输入框不重叠
2. 页面无横向溢出
3. 输入聚焦后主要操作按钮仍可触达
4. 实时助手与模拟面试页的底部控制区不被遮挡

## 4. 控制台与结果记录

检查项：

1. 浏览器控制台无 error
2. 若有 warning，记录是否为既有非阻塞项
3. 记录异常页面、复现步骤、截图与本地地址
