# Web 端交付物

本目录存放 **Web 端验收与线框** 相关材料，可运行的 Web 应用源码仍在仓库根目录（`src/`、`server/`、`public/`、`scripts/`）。

## 目录

| 子目录 | 内容 |
|--------|------|
| `artifacts/` | 桌面端（1280px）截图、验收笔记、Codex 生成图 |
| `wireframes/` | 实时助手、模拟面试线框 HTML、体验入口页 |

## 运行应用

在仓库根目录执行：

```bash
npm install
npm run dev
```

详见根目录 [README.md](../README.md)。
